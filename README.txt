Para correr el proyecto:
	Debe tener la versión de nodejs 7.4.0
	Debe tener MySQL Workbench con el usuario: root y contraseña: 12345
	Correr el script que se encuentra en bd/script.sql
	Abrir la consola, posicionarse en esta carpeta y ejecutar el comando: node server
	Abrir el navegador y poner la dirección localhost:8080
